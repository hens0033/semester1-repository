Notes

Bold
In order to bold a word you have to type b inside angled brackets.
Paragraph
To create a paragraph you have to type p inside angled. If you wish to create an essay you will have to seperate each paragraph.
Nav
A navigation tag or Nav for short, will allow you to create a list of navigation links to other pages on a website.
Footer
A footer tag is for copyright information to be put at the bottom of a page.
Image
In order to put an image into a webpage you will have to link it into the image tag and add alternate text.
